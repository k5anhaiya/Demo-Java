Importing Classes
------------------
Inorder to use those classes that are in package, We will have to write code to import those classes.
we achieve it by following three ways

1)import package.classname;
2)import packagename.*;
3)without import statement.use fully qualified class in code.

without importing classs. Program can not use it.Compiler will give error.

method 1 of importing the code
--------------------------------
 
import javax.swing.JButton;
import javax.swing.JFrame;
import javax.swing.JPanel;
import java.awt.Color;
import java.awt.Font;

public class ImportDemo 
{
	public static void main(String[] args) 
	{
		JFrame fr=new JFrame();
		fr.setSize(400,400);
		fr.setLocationRelativeTo(null);
		JButton bt=new JButton();
		bt.setForeground(Color.blue);
		bt.setFont(new Font("arial",0,22));
		JPanel pa=new JPanel();
		fr.add(pa);
		pa.add(bt);
		fr.setVisible(true);
		
	}

}
