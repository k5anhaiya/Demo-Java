Naming rules
------------
We need to specify name of classes,variables,methods,packages,parameters, constants,etc
Following naming rules must be followed

1)Name must begin with an alphabet or $ or underscore(_)
2)Except $ and _ other symbols are not allowed in name
3)Space is not allowed name
4)Keywords can not be name

Naming conventions
------------------
Following conventions are optional.But it strongly recommened

1)Name should be meaningful
2)Class name should be in TitleCase
3)Method,variable and parameter name should be in camelCase
4)Package name should be in lowercase
5)Constant name should be in UPPERCASE
